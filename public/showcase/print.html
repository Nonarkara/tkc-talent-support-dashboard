1. วัตถุประสงค์

รายงานฉบับนี้จัดทำขึ้นเพื่อสรุปขอบเขตงานที่ผู้รับจ้างได้ดำเนินการพัฒนา ระบบสนับสนุนและพัฒนาทรัพยากรมนุษย์ของบริษัท เทิร์นคีย์ คอมมูนิเคชั่น เซอร์วิส จำกัด (มหาชน) (TKC Digital Twin) ให้แล้วเสร็จตามขอบเขตที่ตกลงไว้ โดยระบบที่พัฒนาขึ้นเป็นแพลตฟอร์มดิจิทัลสำหรับการบริหารและพัฒนาบุคลากร ผ่านกลไกการจัดการทีมแบบ Gamification เพื่อให้ผู้บริหารสามารถมองเห็น ศักยภาพ การจัดสรรทรัพยากรบุคคล และสุขภาพขององค์กรได้ในมุมมองเดียว บนระบบที่ทำงานแบบออนไลน์ตลอดเวลา

2. ขอบเขตการดำเนินงานที่ส่งมอบ
  • 01ระบบ Command Center ออนไลน์ (8 หน้าจอ) — เข้าถึงได้ที่ tkc-digital-twin.fly.dev
  • 02ฐานข้อมูล PostgreSQL บน Cloud (Singapore) — 28 ตาราง ข้อมูลพนักงาน 348 คน
  • 03ระบบ Google Sheets Shadow Mirror — 20 แท็บ sync อัตโนมัติ
  • 04Game Engine — ระบบคะแนน ICA Index, Credo Score, HP/MP/XP
  • 05Formation Engine — ระบบจัดทีมพร้อมทำนายคะแนนทีมและ Budget Cap (Moneyball)
  • 06Ninja Squad Engine — ระบบประเมินความพร้อม Squad สำหรับโปรเจกต์
  • 07Capability Matrix — Heatmap ทักษะเทียบกับความต้องการโปรเจกต์แบบ real-time
  • 08The Tome Printer — ระบบพิมพ์ประวัติการทำงานพนักงานเมื่อครบวาระ (Retirement Ritual)
  • 09Obsidian Export Engine — ส่งออกข้อมูลทีมและพนักงานไปยัง Knowledge Base โดยอัตโนมัติ (348 dossiers)
  • 10Daily Briefing System — รายงานสรุปอัจฉริยะรายวันสร้างขึ้นโดยอัตโนมัติ
